# CitoTemplate
A roblox module loader.

Supported events for controllers:
- Update
- InputBegan 
- InputEnded 
- CharacterAdded 
- CharacterRemoving

Use `shared.Import "MODULE_NAME"` to require modules.

Built in libraries:
- EzShake
- Event (fixed)
- ProfileService
